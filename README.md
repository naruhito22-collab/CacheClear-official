## 🎨 ロゴ設計背景

ロゴは電子回路の「トランジスタ」をモチーフにしています。

| 要素 | 意味 |
|---|---|
| 上横棒（笠木） | VCC電源供給 |
| 中央横棒（貫） | 制御信号（トランジスタゲート） |
| 下部3本の柱 | 出力ピン（ドレイン・ゲート・ソース）。下へ開放し、マルチエージェント構造を表現 |

| 項目 | 値 |
|---|---|
| カラー | `#ff6b35`（Warm Orange） |
| 線幅 | 3.5px |
| アスペクト比 | 1 : 1.3 |
| 推奨背景 | 濃紺（`#001a4d`）／透過 |

初期案は電子回路×鳥居の融合型（シアン `#00d9ff` ＋オレンジ `#ff6b35`）でしたが、最終的にシンプルな「2本の横棒＋3本の縦棒」に整理されたトランジスタ版が確定しました。詳細な検討過程は [concepts.html](concepts.html) / [graphics.html](graphics.html) / [circuit.html](circuit.html) を参照してください。
